# FFXI Character Tracker

https://calm-tor-21934.herokuapp.com/


[FFXI](http://www.playonline.com/ff11us/index.shtml) is a game I've played off and on for **years**. It's vast and complicated and it makes you work for every accomplishment. The number of things you can get involved in can make your head spin, so I decided to create a [Character Tracker](https://calm-tor-21934.herokuapp.com/) to help keep track of everything!

Built with **React**, managed by **Redux**. ♥
